## Getting Started

### 1. Clone this repo
  ```sh
  git clone https://github.com/glprice/python-projects.git
  ```
### 2. Run the scripts with Python 
Make sure you have Python installed on your device (https://www.python.org/)

From the root folder, navigate to the `codecademy_projects` folder in the terminal. 
  ```sh
  cd codecademy_projects
  ```

Run the script. Below is an example of running the `coded_correspondence.py` script.
  ```sh
  python coded_correspondences.py
  ```
### 3. `cheatsheet.md` is my notepad for learning


### Prerequisites

You need Python.
* Python (https://www.python.org/).

Check if it is installed with the following command: 
  ```sh
  python --version
  ```
